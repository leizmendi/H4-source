# H4-source
Fuentes de H4, objetos de Access para importar con Oasis SVN

Para importar los ficheros a una base de datos de Access se debe instalar el complemento para Access de OASIS: https://dev2dev.de/index.php?lang=en e importar los objetos en la base de datos.
La aplicación es un PMS Property Management System para gestión de Campings . Hoteles.
